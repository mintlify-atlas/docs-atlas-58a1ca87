This guide shows basic usage patterns for jsonlogic2sql with real examples from the library.

## Getting started

Create a transpiler and convert JSON Logic to SQL:

```go
import "github.com/h22rana/jsonlogic2sql"

// Create transpiler for your SQL dialect
tr, err := jsonlogic2sql.NewTranspiler(jsonlogic2sql.DialectBigQuery)
if err != nil {
    panic(err)
}

// Transpile JSON Logic to SQL
sql, err := tr.Transpile(`{">": [{"var": "amount"}, 1000]}`)
// Result: "WHERE amount > 1000"
```

## Conditional (IF) statements

### Simple IF

<CodeGroup>
```json JSON Logic
{
  "if": [
    {">": [{"var": "age"}, 18]},
    "adult",
    "minor"
  ]
}
```

```sql SQL Output
WHERE CASE WHEN age > 18 THEN 'adult' ELSE 'minor' END
```
</CodeGroup>

### IF with complex condition

<CodeGroup>
```json JSON Logic
{
  "if": [
    {
      "and": [
        {">=": [{"var": "age"}, 18]},
        {"==": [{"var": "country"}, "US"]}
      ]
    },
    "eligible",
    "ineligible"
  ]
}
```

```sql SQL Output
WHERE CASE WHEN (age >= 18 AND country = 'US') THEN 'eligible' ELSE 'ineligible' END
```
</CodeGroup>

## Using TranspileCondition

Get SQL without the `WHERE` keyword:

```go
tr, _ := jsonlogic2sql.NewTranspiler(jsonlogic2sql.DialectBigQuery)

// Returns: "amount > 1000" (no WHERE prefix)
sql, err := tr.TranspileCondition(`{">": [{"var": "amount"}, 1000]}`)
```
